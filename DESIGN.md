# Design

Визуальная система «Кулинарной книги». Тёплая, уютная, личная — ощущение бумажной тетрадки с рецептами и домашней кухни. Цель: не спутать с дефолтным SaaS, не холодно, не перегружено.

## Theme

**Light, тёплый.** Сцена: девушка на кухне днём или вечером открывает приложение, чтобы приготовить ужин, либо вечером на диване добавляет рецепт. Свет тёплый, настроение спокойное и домашнее. Светлая тёплая тема (кремовая бумага), без тёмного режима в первой версии. Фон — тёплый крем, не белый; текст — тёплый тёмно-коричневый, не чёрный.

## Color

Стратегия: **Committed** — тёплый кремовый фон доминирует на поверхности, терракотовый акцент несёт действия и фокус. Все нейтрали тонированы в тёплый оранжево-персиковый hue. OKLCH везде.

### Палитра (OKLCH)

```
/* Поверхности — тёплая «бумага» */
--bg:            oklch(0.984 0.013 75);   /* основной фон, тёплый крем */
--surface:       oklch(0.975 0.016 72);   /* карточки, поля */
--surface-sunk:  oklch(0.955 0.018 70);   /* вдавленные зоны, чипы */
--border:        oklch(0.90 0.020 68);    /* мягкие тёплые границы */

/* Текст — тёплый тёмно-коричневый, не чёрный */
--ink:           oklch(0.30 0.028 55);    /* основной текст */
--ink-soft:      oklch(0.50 0.024 58);    /* вторичный текст, мета */
--ink-faint:     oklch(0.66 0.020 62);    /* плейсхолдеры, подсказки */

/* Акцент — терракот (действия, фокус, активные табы) */
--accent:        oklch(0.62 0.135 42);    /* терракотовый */
--accent-hover:  oklch(0.57 0.140 41);
--accent-soft:   oklch(0.93 0.045 55);    /* фон акцентных чипов/выделений */
--on-accent:     oklch(0.985 0.013 75);   /* текст на акценте */

/* Семантика, тёплая */
--heart:         oklch(0.60 0.165 25);    /* избранное, мягкий тёплый красный */
--star:          oklch(0.76 0.135 75);    /* оценка, янтарный */
--danger:        oklch(0.58 0.150 28);    /* удаление */
--success:       oklch(0.62 0.090 145);   /* подтверждения, приглушённый */
```

Правила: никогда `#000`/`#fff`. Цвет не единственный носитель смысла (избранное = иконка-сердце + цвет; оценка = звёзды + число). Акцент терракот используется сдержанно — на действиях и активных состояниях, не заливать им всё.

## Typography

Контраст «книжный serif + мягкий sans» создаёт характер тетрадки.

- **Заголовки (`--font-display`)**: `'Lora', Georgia, serif` — тёплый книжный serif с кириллицей. Названия рецептов, заголовки секций, заголовок приложения.
- **Текст (`--font-body`)**: `'Nunito', system-ui, sans-serif` — мягкий округлый sans с кириллицей. Body, подписи, кнопки, поля.
- Загрузка через Google Fonts (Lora: 500/600; Nunito: 400/600/700), `display=swap`.
- Шкала (ratio ≥1.25): 13 / 15 / 17 / 20 / 25 / 32. Body 17px на странице рецепта (читаемость на кухне), 15px в плотных списках.
- Длина строки шагов ≤ 65ch. Межстрочный 1.5 для шагов.
- Иерархия — масштаб + вес (Lora 600 заголовки, Nunito 700 акценты, 400 body), без «плоских» шкал.

## Layout & shape

- Скругления мягкие, тёплые: чипы/кнопки `12px`, карточки/поля `16px`, фото `20px`. Единый «округлый» характер.
- Ритм через разный отступ: щедрый воздух вокруг фото и заголовков рецепта, плотнее в списках ингредиентов. Не одинаковый padding везде.
- Карточки — только там, где это лучший affordance: сетка рецептов на главной (фото — герой). Не оборачивать в карточку всё подряд; страница рецепта — это «лист», а не набор карточек. Никаких вложенных карточек.
- Контейнер контента ≤ `28rem` по центру (мобильный-first, телефон).
- Фото блюда дышат: крупные, скруглённые, заполняют ширину карточки/верх страницы.

## Components

- **Нижняя навигация**: тёплая «бумажная» панель с мягкой верхней границей; активный таб — терракот + чуть жирнее, плюс маленькая иконка над подписью. Тач-цели ≥ 44px.
- **Карточка рецепта**: фото сверху (скругление 20px), под ним название (Lora), строка меты (время · порции) и оценка звёздами; сердце-избранное поверх фото в углу. Плейсхолдер фото — тёплый кремовый с мягкой иконкой блюда, не серый.
- **Кнопки**: главное действие — заливка терракотом, текст `--on-accent`, скругление 12px; вторичное — тёплая обводка `--border` на `--surface`. Опасное (удалить) — текст/обводка `--danger`, не сплошная заливка.
- **Поля ввода**: фон `--surface`, тёплая граница, фокус — кольцо `--accent` 2px; крупные, дышащие, скругление 12-16px.
- **Чипы (категории/теги)**: pill, `--surface-sunk` фон; активный фильтр — `--accent-soft` фон + `--accent` текст. Тонкие, не кричащие.
- **Звёзды / сердце**: `--star` / `--heart`, всегда вместе с формой иконки.
- **Пустые состояния**: дружелюбные, по-человечески («Здесь появятся твои рецепты»), с мягкой иллюстрацией-иконкой и понятной кнопкой действия. «Нет результатов поиска» — отдельный текст от «пока нет рецептов».

## Motion

- Переходы 150-220ms, `ease-out` экспоненциальные (quart/expo). Без bounce/elastic.
- Не анимировать layout-свойства; использовать opacity/transform.
- Микро-отклик: лёгкое затемнение/scale 0.98 при нажатии кнопок и карточек.
- Уважать `prefers-reduced-motion: reduce` — отключать необязательные движения.

## Принципы применения

Тепло важнее плотности (Принцип 1): где обычный продукт был бы сухим — давать воздух и мягкость. Кухонная читаемость (Принцип 2): крупный контрастный текст шагов, большие тач-зоны. Фото и «от кого» — ценность (Принцип 3): не прятать в таблицу. Простота для новичка (Принцип 4): одна задача на экран, дружелюбные подписи.
